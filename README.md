# tree.sh

Простой Bash‑скрипт для среды Bash, создающий древовидное представление структуры каталогов — аналог команды `tree` в Windows CMD.

## Описание

Скрипт `tree.sh` рекурсивно выводит содержимое каталогов в виде иерархического дерева с визуальными разделителями (`├──`, `└──`, `│`). Поддерживает ограничение глубины вывода и отображение скрытых файлов.

## Особенности

* Цветное отображение: папки — синим (жирным), файлы — белым.
* Ограничение глубины дерева с помощью флага `-d`.
* Отображение скрытых файлов (начинающихся с `.`) с флагом `-a`.
* Поддержка путей с пробелами и специальными символами.
* Кроссплатформенность: работает в Linux, macOS и WSL.

## Установка

1. Скачайте скрипт:
   wget https://github.com/VladimirMok/tree-bash/archive/refs/heads/main.zip
   # или клонируйте репозиторий
2. Сделайте его исполняемым:
   chmod +x tree.sh

## Использование

./tree.sh [путь] [-d глубина] [-a]

## Параметры
[путь] — каталог для отображения (по умолчанию: текущая директория .).
-d <число> — ограничить глубину вывода (по умолчанию: 10).
-a — показать скрытые файлы и папки (например, .git, .config).
--help или -h — вывести справку по использованию.

## Примеры

 Вывести дерево текущей директории (глубина до 10 уровней)
./tree.sh
 Вывести структуру каталога /home/user с глубиной до 3 уровней
./tree.sh /home/user -d 3
 Показать скрытые файлы в текущей директории
./tree.sh -a
 Комбинация: глубина 2 + скрытые файлы
./tree.sh ./myproject -d 2 -a
